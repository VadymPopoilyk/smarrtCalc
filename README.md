# smartCalc
This is the repository for our school project SmarrtCalc. The project includes a calculator application that allows you to perform basic mathematical calculations and convert various units.


# Instalation

If you have GitHub CLI:
```Terminal
gh repo clone VadymPopoilyk/smarrtCalc
```
If you don't have GitHub CLI:
```Terminal
git clone https://github.com/VadymPopoilyk/smarrtCalc.git
```
and then login to your GitHub account.

```Terminal
npm install
```
```Terminal
npm run scss
```
